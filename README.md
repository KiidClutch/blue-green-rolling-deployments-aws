# Blue-Green & Rolling Deployments on AWS

## Project Overview

This project demonstrates how to implement zero-downtime deployments on AWS using Blue-Green Deployments and Rolling Deployments with Amazon EC2 Auto Scaling Groups.

## AWS Services Used

- Amazon EC2
- EC2 Auto Scaling
- Launch Templates
- Application Load Balancer (ALB)
- Target Groups
- Lifecycle Hooks
- Warm Pools
- IAM
- Amazon VPC
- Security Groups

## Key Features

- Blue Environment (v1)
- Green Environment (v2)
- Rolling Deployment (v3)
- Zero-downtime traffic shifting
- Instance Refresh
- Lifecycle Hooks
- Warm Pool configuration
